Features

100% Private

Your data never leaves your network.

Lightning Fast

Zero latency access. No cloud, just local.

Use Everywhere

Web, mobile, desktop. Works on any device.

Share Easily

Create secure links and upload dropboxes.

Open Source

Why Nodi?

  • Built for homes and small teams
  • No accounts, no tracking, no ads
  • Lightweight and easy to run
  • Actively developed

Get Nodi running in seconds

One command installer for Linux, macOS and more.

$ curl -fsSL https://get.nodi.sh | bash

Or install manually from our docs.

System Requirements

  • 1 GB RAM minimum
  • Any modern Linux / macOS
  • Docker (recommended)
